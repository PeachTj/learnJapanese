# Learn how to form embedded yes/no questions that begin with a Japanese form of a "wh-" word

**The Focus of This Lesson Is Forming Embedded Wh Questions.**

**Do you know what time it is?**

**何時かわかる*。***

**Formation**

To insert a "wh-" question into another sentence, attach か to the **informal version of the question.**

| **Informal Version of Question** | ***Ka*(か)** | ***Wakaranai.*(わからない。)** | **"English"** |
| --- | --- | --- | --- |
| *Doko ni iku*(どこに行く) | *ka*(か) | *wakaranai.*(わからない。) | "I don't know where I'll go." |
| *Dōshite kanashii*(どうしてかなしい) |  |  | "I don't know why I'm feeling sad." |
| *nani ga suki*(何がすき) |  |  | "I don't know what he/she likes." |
| *Don'na hito*(どんな人) |  |  | "I don't know what kind of person he/she is." |
| *nani*(なに) |  |  | "I don't know what it is." |
| *doko*(どこ) |  |  | "I don't know where it is." |
| *itsu*(いつ) |  |  | "I don't know when it is." |

Please note that the particle **が**should **mark the subject** within the embedded question.

I don't know **what the teacher is saying.**

先生**が*何を言っているか***わかりません。

Can you tell me **what time you're coming to work tomorrow?**

**明日何時に会社に来るか**教えてください。

***Masu* Stem of a Verb + *Ni* + *Iku*/*Kuru***

The *masu* stem of a verb plus the particle *ni* expresses the purpose of movement when followed by motion verbs such as *iku* ("to go"), *kuru* ("to come"), or *kaeru* ("to return").

**For Example:**

1. "to go **to see** a movie"
    
    映画を**みに**行く
    
2. "Since Minoru forgot his homework, he went **to apologize** to the teacher."
    
    実は宿題を忘れたので、先生に**謝りに**行った。
